Contains password hashes.

## Permissions

| Permission | Meaning |
|---|---|
| r | read |
| w | write |
| x | execute |

## Commands Learned

### whoami
Displays current user.

### id
Displays UID and groups.

### ls -la
Displays permissions and hidden files.

### sudo -l
Displays sudo privileges.

### ps aux
Displays running processes.
