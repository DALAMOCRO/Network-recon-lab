# Linux Networking Notes

## Commands Used

### ip a
Displays network interfaces and IP addresses.

### ip route
Displays routing table and default gateway.

### ss -tuln
Displays listening TCP and UDP ports.

### arp -a
Displays ARP cache entries.

## Key Learnings

- Identified local IP address
- Learned how to identify the gateway
- Understood listening vs established connections
- Learned how Linux displays networking information
